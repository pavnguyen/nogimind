# NoGi Mind Content Standard

NoGi Mind treats structured data as the source of truth. Pages, search, graphs, checks, and related links should derive from data instead of one-off page copy.

## Position Standard

- Define top and bottom role goals.
- List danger signals before late escapes.
- Link available skills, common problems, next positions, and related concepts.
- Include body-to-body control points when a position depends on a clamp, wedge, frame, hook, hip line, shoulder line, or knee line.

## Skill Standard

Every important skill should include:

- `quickCard`: goal, exactly three cues, one warning, one if-fail action.
- `microDetailSystem`: five precise details, fast path, left/right guide, troubleshooting, do-not-do list.
- `bodyToBodyDetails`: my body part, opponent body part, side, contact type, force direction, why it works, common misplacement, cue.
- `qualityChecklist`: critical checks first.
- `blackbeltDetails`: clamp, pressure, angle, finish trigger, abort signal, escape prevention.
- `techniqueStateMachine`: role views, realistic outcomes, training progressions, knowledge checks.
- Prerequisites, related skills, related concepts, safety notes when relevant.

## Submission Standard

Submission pages must make safety visible:

- anatomical target
- control requirement
- finishing trigger
- false finish signal
- defender recognition cues
- escape or abort paths
- release protocol
- safety-critical knowledge checks

Use gradual pressure. Release immediately on tap or distress. Do not teach sudden torque, blind rotation, or neck/spine cranking as normal finishing mechanics.

## Modern Technique Standard

Modern or advanced techniques need:

- classification: tier, family, risk, modern system group
- prerequisites
- ruleset notes if relevant
- common counters
- if-fail branches
- safety abort when neck, knee, shoulder, spine, or breathing risk is meaningful

## State Machine Standard

Treat grappling as a state machine:

Position -> Technique -> Outcome -> Next Position / Skill / Submission / Counter / Safety Abort

Outcomes should include at least:

- success
- failure or reset
- counter, branch, or safety abort

Probabilities are optional and only educational. If exact probability is not defensible, use confidence labels.

## Body-To-Body Writing Standard

Every key instruction should answer:

- Which body part of mine?
- Which body part of the opponent?
- Which side?
- What contact type?
- What force direction?
- Why does it work?
- What common misplacement breaks it?
- What short cue fixes it?

Good:

- “My right hand pulls the opponent’s left wrist across centerline.”
- “My chest drives diagonally toward the opponent’s far hip.”
- “My choking elbow aligns under the opponent’s chin before squeezing.”

Weak:

- “Control the arm.”
- “Apply pressure.”
- “Finish the choke.”
