# Duplication And Cleanup Report

## Repeated Patterns Found

- Control before submission.
- Angle before force.
- Clamp before torque.
- Hand fight before choke.
- Protect knee line.
- Hide heel before rotation.
- Pressure needs direction.
- Frames are structure.
- Reactions create branches.

## Cleanup Decision

The broad explanations stay in shared concepts and shared knowledge references. Skill content now keeps only the local application: exact body contact, line, cue, failure signal, and branch.

## What Changed

- Armbar, Triangle, and Omoplata now use skill-specific micro-details instead of generic submission filler.
- Search aliases were expanded instead of duplicating long concept paragraphs into every skill.
- Related skills in modern expansion content were remapped from planned placeholders to implemented skills.
